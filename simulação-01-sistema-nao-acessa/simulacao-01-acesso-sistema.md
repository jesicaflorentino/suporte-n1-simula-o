# Chamado 01 – Erro de Conexão ao Sistema

## Descrição do chamado
**Relato do usuário:**  
"Não consigo acessar o sistema, aparece erro de conexão."

---

## 1. Triagem inicial
Perguntas realizadas para entendimento do problema:

- Desde quando o erro ocorre?
- Outros usuários conseguem acessar o sistema?
- A internet funciona normalmente em outros sites?
- Houve alguma alteração recente no ambiente?  
  (troca de senha, atualização do sistema, queda de energia)

---

## 2. Verificações solicitadas ao usuário
Orientações iniciais para coleta de informações:

- Testar conexão com a internet
- Verificar data e hora do sistema
- Testar acesso ao sistema em outro navegador
- Testar login em outro equipamento (quando possível)

---

## 3. Diagnóstico
Após as verificações, foi constatado que:

- O sistema estava funcionando normalmente para outros usuários
- A conexão com a internet estava ativa
- O computador do usuário estava com **data e hora desatualizadas**

**Causa identificada:**  
Falha de autenticação causada por divergência de data e hora do sistema.

---

## 4. Solução aplicada
Orientações fornecidas ao usuário:

- Realizar sincronização manual de data e hora
- Ativar configuração automática de data e hora
- Reiniciar o navegador
- Testar novamente o acesso ao sistema

---

## 5. Resultado
- Acesso ao sistema normalizado
- Usuário confirmou funcionamento correto

---

## 6. Encerramento do chamado
Chamado encerrado com orientação preventiva para manter a sincronização automática de data e hora ativada no sistema operacional.
