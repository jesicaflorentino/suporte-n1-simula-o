# Suporte Técnico N1 – Simulações de Atendimento

Este repositório reúne simulações práticas de atendimentos de Suporte Técnico N1, com foco em:

- Triagem correta do problema
- Comunicação clara com o usuário
- Identificação da causa raiz
- Aplicação de soluções simples e eficazes
- Boas práticas de encerramento de chamados

O objetivo é demonstrar raciocínio técnico, organização e postura profissional em atendimentos de primeiro nível.

## Estrutura

- `/atendimentos`  
  Contém os registros detalhados de chamados simulados, organizados por cenário.
